# Helpers

In addition to all Bootstrap utility classes, we implemented these ones:

## Shadows

- `.shadow-to-right`: adds a drop shadow pointing to the right
- `.shadow-to-left`: adds a drop shadow pointing to the left
- `.shadow-to-bottom`: adds a drop shadow pointing to the bottom
- `.shadow-floating`: adds an even drop shadow

## Background

- `.bg-blue`, `.bg-primary-light`: sets a colored background
- `.bg-gradient`: sets a great gradient background
